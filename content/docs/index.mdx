In this section, you will find information on how to use the wiki and what you can use to make the wiki great. If you are new to the wiki, this is the **perfect** place to be as we will guide you through our entire project step by step.

If you are not new to the wiki or would just like to explore at your own pace feel free to use the navigation bar on the left to explore at your lesiure.

_Now lets talk about the features of this wiki._

## Tables

Tables are widely used in documentation and are **100%** supported by the wiki. You can create a table by using the following syntax:

| Column 1 | Column 2 | Column 3 |
| -------- | -------- | -------- |
| R1C1     | R1C2     | R1C3     |
| R2C1     | R2C2     | R2C3     |
| R3C1     | R3C2     | R3C3     |
| R4C1     | R4C2     | R4C3     |

## Steps

Steps which are are a great way to break up your documentation into steps. There are two ways you can use steps:

**Steps with headings**

Steps with headings will display numbers
